Elasticsearch 版本edit

本书的初始印刷版针对的是 Elasticsearch 1.4.0，不过我们一直在不断更新内容和完善示例 本书的线上版本 针对的是 Elasticsearch 2.x。

你可以访问这本书的 GitHub 仓库 来追踪最新变化。
