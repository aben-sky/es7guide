ids

根据文档的id返回文档。 该查询使用存储在 _id 字段中的文档id。

请求示例 

GET /_search
{
    "query": {
        "ids" : {
            "values" : ["1", "4", "100"]
        }
    }
}

ids的顶级参数

values
(必需, 字符串数组) 文档id 数组。
