Elasticsearch version 7.1.0edit

Also see Breaking changes in 7.1.

Enhancementsedit

Security
  • Moved some security features to basic. See 7.1.0 Release highlights.
Authentication
  • Log warning when unlicensed realms are skipped #41778
Infra/Settings
  • Drop distinction in entries for keystore #41701

Bug fixesedit

Cluster Coordination
  • Handle serialization exceptions during publication #41781 (issue: #41090)
Infra/Core
  • Fix fractional seconds for strict_date_optional_time #41871 (issue: #41633)
Network
  • Enforce transport TLS on Basic with Security #42150
Reindex
  • Allow reindexing into write alias #41677 (issue: #41667)
SQL
  • SQL: Fix issue regarding INTERVAL * number #42014 (issue: #41239)
  • SQL: Remove CircuitBreaker from parser #41835 (issue: #41471)
Search
  • Fix IAE on cross_fields query introduced in 7.0.1 #41938 (issues: #41125, #41934)
