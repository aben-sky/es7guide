Generating SP metadataedit

Some Identity Providers support importing a metadata file from the Service Provider. This will automatically configure many of the integration options between the IdP and the SP.

The Elastic Stack supports generating such a metadata file using the bin/elasticsearch-saml-metadata command in your Elasticsearch directory.

The documentation for the elasticsearch-saml-metadata utility describes how to run it, and the available command line options.
