安装X-Pack (Set up X-Pack)

X-Pack 是 Elastic Stack 的一个弹性的堆栈扩展，提供安全、警报、监控、报告、机器学习和许多其他功能。 默认情况下，安装 Elasticsearch 时，会安装 X-Pack。

如果你想尝试 X-Pack 的所有功能，你可以 开始30天的试用。 试用期结束时，你可以购买订阅以继续使用 X-Pack 组件的全部功能。 更多信息请参考 https://www.elastic.co/subscriptions。

  • 配置安全(security)
  • 配置 X-Pack Java 客户端
